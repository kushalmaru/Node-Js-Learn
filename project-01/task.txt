REST API - JSON

GET /users - HTML Doc Render
GET /api/users - List all users JSON

GET /api/users/1 - Get the users with ID 1
GET /api/users/2 - Get the users with ID 2

POST /api/users - Create new user

PATCH /api/user/1 - Edit the users with ID 1

DELETE /api/users/1 - Delete the users with ID 1
